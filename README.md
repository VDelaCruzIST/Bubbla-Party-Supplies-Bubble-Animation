# Bubbla Party Supplies Bubble Animation

## Overview
This repository contains an animated party supplies website developed as part of my coursework while pursuing a Bachelor of Business Administration in Information Systems and Technology (IST).

The project demonstrates advanced JavaScript programming concepts through the creation of animated bubbles that dynamically appear, move, rotate, change color, and fade away over time. The application utilizes object-oriented programming, animation loops, timers, randomization, and DOM manipulation to create an interactive visual experience.

## Features
* Dynamic bubble animation system
* Object-oriented JavaScript design
* Random bubble generation
* Bubble movement and collision detection
* Bubble rotation effects
* Dynamic color shifting
* Fade-out animation effects
* Real-time DOM manipulation
* Animated visual user experience
